# Login Authentication System

A full stack MERN authentication system with complete frontend, backend, API routes, database integration and user authentication architecture.

## Features

- User Login System  
- User Signup System  
- Forgot Password Functionality  
- Dashboard Access  
- Authentication Middleware  
- API Route Handling  
- MongoDB Database Integration  
- Backend Architecture using Express.js  

## Tech Stack

Frontend  
- HTML  
- CSS  
- JavaScript  
- React.js  

Backend  
- Node.js  
- Express.js  

Database  
- MongoDB  
- Mongoose  

## Project Architecture

client/ → React Frontend Components  
server/ → API Routes and Controllers  
models/ → Database Schema  
database/ → Database Connection  
frontend/ → Initial Frontend Version  
backend/ → Initial Backend Version  

## Purpose

Designed as a full stack authentication system demonstrating MERN stack architecture and scalable backend structure.
